# 📊 Data Analysis Portfolio

Welcome to my **Data Analysis Portfolio** repository!

This repository contains my journey of learning and practicing data analysis. It includes datasets, exploratory data analysis (EDA), data cleaning exercises, visualization projects, statistical analysis, and small real-world tasks completed while improving my analytical skills.

The goal of this repository is to document my progress, build a strong portfolio, and demonstrate practical applications of data analysis using Python and its ecosystem.
